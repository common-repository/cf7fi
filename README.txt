The Plugin based on Contact Form 7 and advance the functions for Formisimo integration.

== Description ==

Formisimo is a nice service for tracking your input of form.
It helps for increase the convertion rate.
This plugin helps integrate the tracking and conversion Code into the Contact 7 Form Plugin.

Beta Registration for Form Tracking under [Formisimo](http://www.formisimo.com/ "get your Account!")
  
Plugin support is available on [Google+](https://plus.google.com/100640944983138345674?rel=author "Add me on Google+")

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the pluginfolder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Plugins -> Settings and put your tracking Code in
1. Fill out the new field "Formisimo" in your form
1. analyse your form and get happy
